# 大阪おすすめ居酒屋MAP 〜YouTube居酒屋動画から〜

YouTubeの大阪居酒屋動画 10本から洗い出した梅田・駅前ビル・北新地・難波・天満・京橋・新世界・玉造・新大阪などの名店 29軒を、
エリア別に色分けした地図＋一覧にまとめた単一HTMLサイト。
スポットをクリックすると、実際に紹介している動画（YouTube埋め込み）がモーダルで見られる。
各店の**日曜日の営業時間**を調査・掲載しており、日曜の営業状況（昼から営業／夕方〜営業／定休）でフィルタできる。

公開URL: https://oneliner22.github.io/osaka-izakaya-map/

## 構成

- `index.html` — 生成物（データ埋め込み済み単一ファイル、file:// でも動く）
- `template.html` — テンプレート（Leaflet + markercluster、YouTube埋め込みモーダル、日曜営業フィルタ）
- `build_spots.py` — スポット定義（エリア/ジャンル/座標/説明/日曜営業/出典動画ID）→ `data/spots.json`
- `build_html.py` — `data/*.json` + `template.html` → `index.html`
- `data/videos.json` — 出典YouTube動画（ID → タイトル/チャンネル）
- `data/config.json` — タイトル・リード文・地図中心/ズーム等

## ビルド

```
python build_spots.py
python build_html.py
```

## 出典

スポット情報は以下のYouTube動画に基づく（動画の著作権は各チャンネルに帰属）:
じわじわクセになる世界 / しゅんちゃん【大阪グルメ】 / うに酒ボーイあか / らりまるのお酒日記 /
見取り図ディスカバリーチャンネル / こんちゃん。 / Yunaty日本語 / 必殺おでかけ人 /
ハングリーズ【関西・大阪グルメ】 / 瓶ビール班長の旅ログ

座標は国土地理院ジオコーディングAPI・食べログ・公式サイト等で裏取り（番地レベル）。
日曜の営業時間は2026年7月調査時点（食べログ/ホットペッパー/公式サイト等。変更の可能性があるため訪問前に要確認）。
「スタンドB」は動画内で所在地の明言がなく、条件に合致する唯一の店舗（高槻）を「およその位置」として掲載。
